ブロギルラジオタイムフリーに参加するには、キャスター登録が必要です。
放送する人はパーソナリティと呼ばれますが、混同を避けるためにキャスターという言葉を使います。
パーソナリティ権限をお持ちの方はclifeにDM、メンションいただければ、キャスター登録と解除はいつでもできます。

録音開始条件
1　キャスター登録済
2　誰も録音していない（ブロギルラジオに録音ボットがいない）
3　マイクミュート解除
以上を満たした状態でブロギルラジオに接続すると録音開始

録音終了
・ブロギルラジオを切断する
※後続のラジオを聴くために接続状態を継続すると、マイクをミュートにしても録音継続される。
　他のキャスターが録音できないので、いったん切断してマイクミュートにしてから再接続する。
　再接続してからマイクミュートを解除すれば、後続番組でゲストとしてコラボできる。　
　
キャスターのコラボ
・録音ボットは同時に複数の番組を録音できず、最初に録音開始したキャスターの番組にゲストとして全ての音声が録音される。
・録音番組を切り替えるには新旧キャスターがいったん切断して、新キャスターが先に再接続する。
